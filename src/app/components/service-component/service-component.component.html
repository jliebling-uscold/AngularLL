Services

  • A service is a class that provides a specific functionality or a set of related functionalities that can be shared across different components or modules in your application.
  • Services are typically used to interact with a server, a database, or third part applications in general.

  • In USCS-FE we split REST calls into a serperate Async services

  • Services are also used to share data between different components. (ie: Layer Pick Task screen and Advanced Window Selection)
  • Services are defined using the @Injectable decorator and can be injected into components or other services using the Angular dependency injection system.
